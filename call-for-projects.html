STANDCON 2025

Call For Projects (CFP)

Share your cybersecurity project at STANDCON 2025! We're now accepting student submissions for our Student Project Corner showcase.

Selected projects will be featured at our in-person conference on 6 and 7 March 2025. If your project is chosen, you'll receive:

- Full conference access

- A dedicated 1-hour presentation slot to showcase your work

Submit your cybersecurity project today!

Submission Requirements

  1. Projects submitted must not be commercial products.
  2. Submit the respective details onto the google form above:
    1. Project title
    2. Project description (max 400 words).
    3. Project members (max 3 members)
      1. Member's name
      2. Member's school/organisation
      3. Member's email
      4. Member's short biography (max 100 words)
      5. Link to socials (if available)
    4. Link to project code with project setup and usage instruction (Eg. Github)
    5. Project deck (optional)

Project Evaluation

  1. Relevance
    1. A project that addresses a new or emerging cybersecurity threat.
    2. A project that applies cybersecurity principles to a new or innovative application.
    3. A project that demonstrates the use of cybersecurity technologies to solve a real-world problem.
  2. Innovation
    1. A project that proposes a new approach to cybersecurity that has not been previously implemented.
    2. A project that improves upon an existing cybersecurity solution.
    3. A project that demonstrates the use of new or emerging technologies to improve cybersecurity.
  3. Technical Merit
    1. A project that is well-designed and implemented.
    2. A project that is free of errors or vulnerabilities.
  4. Communication
    1. A project that is well-documented and easy to understand.
    2. A project that is presented in a clear and concise manner.
    3. A project that is original and interesting
